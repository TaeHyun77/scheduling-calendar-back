# scheduling-calendar-project
캘린더를 통한 일정 관리 프로젝트

# 목적 
나만의 캘린더 일정 관리 페이지를 만들어 보고자 제작

# 사용 기술
- BackEnd : SpringBoot, Spring Data Jpa
- FrontEnd : React.js
- DataBase : MySQL

# 기능
- Oauth를 통한 카카오, 구글, 네이버 로그인
- React.js의 FullCalendar와 DatePicker 모듈을 활용한 Calendar 구현
- 일정 저장, 수정, 삭제
- 작성한 일정 확인
